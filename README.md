# Predicting Donor Retention for NayePankh Foundation
### Machine Learning Internship — Selection Task

A machine learning project that predicts whether a **first-time donor will donate again within 12 months**, and explains **which factors drive repeat giving** — so the NGO can focus limited retention outreach where it matters most.

> ⚠️ **Data is fully synthetic.** No real or proprietary donor data is used. The dataset is generated by the notebook itself for demonstration. The pipeline is exactly what would be applied to NayePankh's real data once collected.

---

## 🎯 Objective

Keeping an existing donor is far cheaper than finding a new one, yet most first-time donors never give again. This project builds a model to predict repeat giving and surface its drivers, turning donation records into an actionable retention tool.

## 🧠 What the Notebook Does

1. **Introduction & objective** — the NGO retention problem
2. **Data loading** — 3,000 synthetic first-time donors
3. **Exploratory data analysis** — distributions, repeat rates, visual patterns
4. **Preprocessing & feature engineering** — cleaning + one-hot encoding via a scikit-learn pipeline
5. **Train/test split** — stratified 80/20
6. **Model comparison** — Dummy baseline → Logistic Regression → Random Forest → XGBoost
7. **Evaluation** — confusion matrix, ROC curve, precision/recall/F1/AUC
8. **Explainability** — feature importance + SHAP (direction of each effect)
9. **Recommendations** — plain-language actions for the Foundation
10. **Limitations & next steps** — honest caveats

## 📊 Dataset

Each row is one first-time donor. Target = `donated_again` (1 = gave again, 0 = did not).

| Feature | Description |
|---|---|
| `first_gift_amount` | First donation size (INR) |
| `donation_channel` | UPI / Card / Cash / BankTransfer |
| `campaign_source` | SocialMedia / Event / Website / Referral |
| `city` | Donor location |
| `donor_age_band` | Age group |
| `first_gift_month` | Month of first gift (1–12) |
| `is_festival_season` | 1 if first gift in Sep–Nov |
| `communication_optin` | 1 if agreed to receive updates |
| `time_on_first_visit_secs` | Time on site, first visit |
| **`donated_again`** | **Target** |

## 📈 Results (on synthetic data)

Four models were compared. Because most donors repeat (62%), **accuracy alone is misleading** — a model that blindly predicts "everyone repeats" scores deceptively well. We therefore judge on **F1** and **ROC-AUC**.

| Model | Accuracy | F1 | ROC-AUC |
|---|---|---|---|
| Dummy (baseline) | 0.62 | 0.76* | 0.50 |
| Logistic Regression | 0.64 | 0.75 | 0.64 |
| **Random Forest** | **0.65** | **0.76** | **0.64** |
| XGBoost | 0.61 | 0.71 | 0.61 |

*\*The baseline's high F1 is exactly the trap the notebook explains: it predicts "repeat" for everyone, so it never misses a repeater (recall = 1.0) but has no real skill (AUC = 0.50).*

The tree and linear models clearly beat the baseline on **ROC-AUC** (0.64 vs 0.50), confirming there is **genuine, learnable signal** — while the modest absolute scores honestly reflect that real donor behaviour is noisy and hard to predict perfectly.

## 🔑 Key Findings

- **Communication opt-in** is one of the strongest predictors of repeat giving.
- **Referral and event** donors are more loyal than one-off social-media donors.
- **Mid-sized** first gifts repeat more than very small or very large ones.
- **Festival-season** first gifts are linked to higher repeat rates.
